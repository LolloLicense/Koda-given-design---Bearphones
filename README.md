Ett litet skolprojekt där jag övade på att **koda en given design** (layout + detaljer) med fokus på ren HTML/CSS och lite JavaScript för interaktion.

## Struktur
- `index.html` – sidan
- `style.css` – styling
- `assets/` – bilder & ikoner
- `js/` – scripts
